# One Bite

A polyphonic step-sequencer synthesizer built with [Faust](https://faust.grame.fr/) and the [JUCE](https://juce.com/) framework. Features 4 pattern tracks, per-track controls, distortion, and a custom dark-themed UI.

## Building

### Prerequisites

| Platform | Requirements |
|----------|-------------|
| **Windows** | Visual Studio 2019 or 2022 (with C++ Desktop workload), CMake 3.22+, Git |
| **macOS** | Xcode 13+, CMake 3.22+, Git |
| **Linux** | GCC 9+ or Clang 10+, CMake 3.22+, Git, ALSA (`libasound2-dev`), X11 (`libx11-dev`), Xext, Xinerama, Xrandr, Xcursor, FreeType (`libfreetype6-dev`), WebKit2GTK (`libwebkit2gtk-4.0-dev`), cURL (`libcurl4-openssl-dev`) |

JUCE 8.0.4 is fetched automatically during configuration — no manual JUCE installation required.

### Windows

```bash
cmake -B build -G "Visual Studio 16 2019" -A x64
cmake --build build --config Release
```

For Visual Studio 2022, use `-G "Visual Studio 17 2022"` instead.

### macOS

```bash
cmake -B build -G Xcode
cmake --build build --config Release
```

Or with Unix Makefiles:

```bash
cmake -B build
cmake --build build --config Release
```

### Linux

```bash
# Install dependencies (Debian/Ubuntu)
sudo apt-get install -y libasound2-dev libx11-dev libxext-dev libxinerama-dev \
    libxrandr-dev libxcursor-dev libfreetype6-dev libwebkit2gtk-4.0-dev libcurl4-openssl-dev

cmake -B build
cmake --build build --config Release
```

### Build Artifacts

After a successful build, artifacts are located at:

```
build/one_bite_artefacts/Release/
    VST3/       # VST3 plugin
    Standalone/  # Standalone application
```

On macOS, an AU (Audio Unit) plugin is also produced.

## Features

- **4 Pattern Tracks** — Track A (16 steps), Track B (8 steps), Track C (4 steps), Track D (2 steps)
- **Per-Track Controls** — Volume, pan, random, clear
- **Clock Modes** — BPM, Hz, and sample-rate driven
- **Distortion** — Drive, dry/wet mix, bypass
- **Gate** — Momentary trigger control
- **16-Voice Polyphony** — Full MIDI support with velocity sensitivity
- **Custom UI** — Dark-themed interface with color-coded tracks, arc-style knobs, and clickable step pads

## Architecture

The DSP engine is generated by **Faust 2.81.10** from a `.dsp` source file using the `juce-plugin.cpp` architecture. The entire DSP and plugin framework is compiled into a single self-contained C++ file (`one_bite/FaustPluginProcessor.cpp`). A custom JUCE editor (`OneBiteEditor`) provides the user interface.

### Parameter Map

| Group | Parameters |
|-------|-----------|
| Clock | mode (BPM/Hz/Sample), rate |
| Track A | vol, pan, rnd, clr, 16 step toggles |
| Track B | vol, pan, rnd, clr, 8 step toggles |
| Track C | vol, pan, rnd, clr, 4 step toggles |
| Track D | vol, pan, rnd, clr, 2 step toggles |
| Distortion | drive, dry/wet, bypass |
| Global | gate |
